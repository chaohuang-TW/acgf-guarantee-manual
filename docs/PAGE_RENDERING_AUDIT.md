# 頁面呈現模式盤點

本文件記錄中華民國115年4月《保證業務作業手冊（農漁會版）》全203個PDF實體頁面的視覺盤點。盤點以低解析聯絡表逐頁檢視，再依正式PDF確認特殊頁面；暫存縮圖不納入版本庫。

## 三種呈現模式

- `text`：一般正文沿用PDF既有文字層，以HTML文字顯示。
- `source-preview`：複雜表格、查索表、正式書表或沒有可用文字層的指定頁面，以完整原PDF頁面WebP預覽顯示；有文字層時仍在預設收合區保留原文字。
- `blank-page`：經人工確認的原始空白頁只顯示空白頁說明，不產生圖片。

## 必處理頁面

| 規則 | 印刷頁／PDF頁 | 判定理由 |
| --- | --- | --- |
| 保證對象及資格認定表 | 印刷1–2／PDF 9–10 | 多欄、合併儲存格及跨頁表格，純文字無法保存欄列關係。 |
| 最高保證成數表 | 印刷6／PDF 14 | 數值對照表，欄位位置直接影響成數解讀。 |
| 附錄十七 | 印刷111／PDF 121 | 大型範圍對照表，含大量框線與分類欄。 |
| 附錄十八 | 印刷112–116／PDF 122–126 | 複雜數字查索表；PDF 126沒有可用文字層。 |
| 信用保證書表及專用書表 | 印刷117–186／PDF 129–186、189–196、199–202 | 正式申請書、勾選欄、簽章欄與位置式欄位必須保存原版面。 |
| 沒有可用文字層的指定頁 | PDF 56、126、128、188、198 | 依設定直接顯示原頁預覽，不以OCR或推測文字補充。 |
| 原始空白頁 | PDF 2、4 | 已逐頁確認為來源文件空白頁。 |

同一頁命中多個`source-preview`規則時只產生一張預覽；PDF 126即為此情形。

## 額外人工判定

- 印刷第20頁／PDF第28頁「農業信用保證基金保證手續費率表」：全頁為數值對照表，文字模式無法可靠保留各期別、案件類別與費率的欄列對應，因此新增`guarantee-fee-rates`規則。

其餘一般正文即使含簡單條列或少量框線，仍維持`text`，避免不必要地把可閱讀、可複製文字轉成圖片。

## 盤點結果

- `text`：117頁。
- `source-preview`：84頁。
- `blank-page`：2頁。
- 合計：203頁。

本次未重建任何HTML表格。數字查索表與正式書表的欄列關係依賴原始位置，若以空白字元或自動規則猜測，可能造成實質誤讀。也未使用OCR；沒有文字層的頁面直接保留原始版面，避免金額、日期、成數與格式編號誤辨。

後續若要將表格重建成HTML，必須逐頁、逐欄、逐列與正式PDF人工核對，另行驗證跨欄標題、單位、註腳、金額、百分比及頁面延續關係；在核對完成前，原PDF頁面預覽仍是正式呈現依據。
